-> 函数的书写方式上 含有区别
    - es6 创建函数
        1)（a） => console.log(a);
    - es5 创建函数
        1) function fn(a) { console.log(a) } ;
    - 在函数书写的方式 产生了 区别
        1) 在其他的 区别 那么就是  es6 的函数创建方式 不会产生作用域的问题
        2) 但是 在es5 中 是会产生 作用域的 可以限制变量的
    - 在es6 中 可以 进行书写 常量的 使用 count 关键词
        1) count str = 'string';
        2) 与使用 var 含有 声明区别呢  那么就是 count 声明的是 常量 在赋值 之后 不能 在进行赋值啦
    - es6 中 可以 进行 声明 另一种形式的变量 该变量会被 语句作用域 进行限制 作用范围
        1) let num = 10;  --- 使用这样 的方式 进行的变量 那么会被 语句作用域 进行限制的
